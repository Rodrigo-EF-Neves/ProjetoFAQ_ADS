Perguntas frequentes

HTML e CSS são linguagens de programação?

HTML e CSS não são linguagens de programação porque não possuem recursos para controle de lógica como variáveis ou interatividade avançada. O HTML é uma linguagem de marcação e o CSS é uma linguagem de estilo.

Qual a diferença entre front-end e back-end?

O front-end é a parte de uma aplicação que é visível e interativa para o usuário, contendo elementos como botões, caixas de seleção, mensagens de texto, gráficos, etc. Por outro lado, o back-end é a parte que o usuário não pode ver como os dados e a infraestrutura que é executada pelo servidor.

Qual foi a primeira lingugem de programação criada?

A primeira linguagem de programação foi o Fortran. A lingugem, criada em meados da década de 1950, foi uma das primeiras tecnologias a oferecer recursos como loops e sub-rotinas no mundo da programação.

Qual é a origem do JavaScript?

A primeira implementação do JavaScript foi criada por Brendan Eich na empresa Netscape Communication Corporation. O JavaScript foi concebido para rodar no servidor, mas isso mudou com o passar do tempo já que, hoje em dia, o JavaScript é uma linguagem Client-side.

O que faz o comando TRUNCATE no SQL?

O comando TRUNCATE, assim como o o comando DELETE, é usado para excluir registros de uma tabela. O TRUNCATE é usado para remover todas as linhas de uma tabela, liberando todo o espçao em disco alocado para uma tabela.
